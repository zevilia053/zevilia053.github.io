<h2>IT Infrastruktur</h2>

<p>
IT infrastruktur adalah dasar dari semua sistem teknologi informasi.
Sistem ini terdiri dari perangkat, jaringan, dan aplikasi
yang digunakan untuk mengelola data.
</p>

<p>
GitHub digunakan sebagai tempat penyimpanan source code
dan menjadi bagian dari IT infrastruktur modern.
</p>

<h3>IT Infrastruktur dan Security</h3>
<p>
Security sangat penting untuk melindungi data dan sistem
dari akses yang tidak sah.
</p>

<h3>Perkembangan IT Infrastruktur</h3>
<p>
Perkembangan teknologi membuat sistem IT berpindah
dari komputer lokal ke cloud dan GitHub.
</p>

<h3>Aset IT</h3>
<p>
Aset IT meliputi hardware, software, dan jaringan
yang semuanya memiliki risiko masing masing.
</p>
